Toasts

Notificações temporárias via ToastService — aparecem no canto superior direito e somem após 4 segundos

Disparar toasts

Toast de Sucesso Toast de Erro Toast de Aviso Toast de Info

Como usar

// Injete o ToastService no seu componente

private readonly

toast = inject(ToastService);

// Dispare toasts de qualquer lugar

this.toast.show({ message: 'Usuário salvo!', type: 'success' });

this.toast.show({ message: 'Erro ao salvar.', type: 'error' });

this.toast.show({ message: 'Aviso importante.', type: 'warning', duration: 6000 });
