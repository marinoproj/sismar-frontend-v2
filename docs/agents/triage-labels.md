# Labels de triage

As skills falam em termos de cinco papéis canônicos de triage. Este arquivo mapeia esses papéis para as strings de label usadas no issue tracker deste repositório.

| Label neste projeto      | Label no nosso tracker | Significado                                        |
| -------------------------- | -------------------- | -------------------------------------------------- |
| `needs-triage`             | `needs-triage`       | Maintainer precisa avaliar esta issue              |
| `needs-info`               | `needs-info`         | Aguardando o reporter por mais informações         |
| `ready-for-agent`          | `ready-for-agent`    | Totalmente especificada, pronta para agente AFK    |
| `ready-for-human`          | `ready-for-human`    | Requer implementação humana                        |
| `wontfix`                  | `wontfix`            | Não será tratada                                   |

Quando uma skill mencionar um papel (ex.: "aplicar a label de triage AFK-ready"), use a string de label correspondente nesta tabela.

Edite a coluna da direita para corresponder ao vocabulário que você de fato usa.
